# Cluster-Beowulf
Cluster Beowulf para algoritmo de procesamiento de imágenes.

Aplica el filtro Kuwahara.

## Dependencias
Requiere OpenCV 3.4.1.

Para instalar OpenCV ejecutar el script config.sh

Dar permisos: 

$ sudo chmod +x config.sh

Y ejecutarlo: 

$ ./config.sh
